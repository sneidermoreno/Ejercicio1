HTML

HTML son las siglas de HyperText Markup Language. Es el lenguaje de marcado principal para la creación de páginas web.

Creado por Tim Berners-Lee, HTML ha evolucionado significativamente desde sus inicios.

Historia de HTML

Los inicios del lenguaje HTML se remontan a principios de la década de 1990. Desde entonces, ha pasado por varias versiones, cada una añadiendo nuevas funcionalidades y mejoras.

Versiones de HTML

Tim Berners-Lee definió la primera versión de HTML en el año 1991. En la actualidad, la última versión de HTML es HTML5.
